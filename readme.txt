Searches all content for "mailto:" links, then unicodes them to protect from spiders. No Javascript required!

== Description ==

Spam is one of the most frustrating things about the internet.
And, even more frustrating is how spiders scrape blogs and other sites for email addresses to spam.
Wouldn't it be nice if you could put mailto links just how you were taught, without fear of spiders?
Well now you can! With this plugin, all mailto links will be filtered on display and converted to their unicode counterparts.

== Installation ==

About as easy as 1,2,3 (ok, easier!)

1. Upload the `dbd-mailto-encoder` directory to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enjoy peace of mind as the plugin instantly goes to work!

== Frequently Asked Questions ==

= What does a standard mailto link look like? =

`<a href="mailto:email@domain.com">email me</a>`

= What does a unicoded version look like? =

`<a href="&#109;&#97;&#105;&#108;&#116;&#111;&#58;&#101;&#109;&#97;&#105;&#108;&#64;&#100;&#111;&#109;&#97;&#105;&#110;&#46;&#99;&#111;&#109;">email me</a>`

= What is a spider? =

A spider is a small program that navigates the 'web' looking for something, in this case email addresses.
